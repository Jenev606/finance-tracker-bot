# Бот "Фінансовий трекер студента"

## Мета проєкту
Розробити консольного бота на мові Python, який допомагає користувачеві (студенту) зручно вести облік особистих витрат, контролювати загальний бюджет та аналізувати витрати за різними категоріями. 

## Опис логіки та алгоритму роботи бота
Програма працює у безперервному циклі, очікуючи команди від користувача. 
**Алгоритм роботи:**
1. При запуску програма перевіряє наявність файлу з даними. Якщо файл існує, дані завантажуються в оперативну пам'ять; якщо ні — створюється порожня структура.
2. Бот виводить вітальне повідомлення.
3. Користувач вводить команду.
4. Програма розпізнає команду та викликає відповідну функцію (додавання витрати, встановлення бюджету, фільтрація тощо).
5. Якщо команда передбачає зміну даних (наприклад, додавання витрати), дані одразу автоматично зберігаються у файл.
6. При додаванні витрати алгоритм перевіряє суму загальних витрат. Якщо вона перевищує бюджет, виводиться попередження.
7. Цикл повторюється до моменту введення команди "вийти".

## Опис структури даних
Дані зберігаються у форматі JSON у файлі `tracker_data.json`. 
Кореневий об'єкт містить два ключі:
**budget:** число з плаваючою крапкою (float), яке зберігає встановлений бюджет.
**expenses:** масив (list) словників (dict). Кожен словник описує окрему витрату і містить ключі:
- **amount:** сума витрати (float).
- **category:** категорія витрати (string).
- **date:** дата у форматі РРРР-ММ-ДД (string).
- **comment:** короткий коментар (string).

## Інструкція зі запуску програми
1. Переконайтеся, що на вашому комп'ютері встановлено Python 3.
2. Завантажте файл `bot.py` в окрему папку.
3. Відкрийте термінал (або командний рядок) у цій папці.
4. Запустіть програму командою: `python bot.py` (або `python3 bot.py` для macOS/Linux).

**Користувач:** 200

**Користувач:** Обід в їдальні
**Бот:** Витрату додано успішно!

**Користувач:** залишок
